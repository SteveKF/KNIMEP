MarvinSpace

MarvinSpace is a 3D visualization tool for small molecules, proteins, nucleic acids or crystals. Using MarvinSpace you can also visualize various molecular surfaces, molecular orbitals as well as volumetric data such as electrostatic potential, hydrophobicity. Manual editing in 3D is also supported.

Dialog Options

The column in the input data table containing structure in Structure(Mrv, Sdf, Smiles, etc.) format, or protein in PDB format.
