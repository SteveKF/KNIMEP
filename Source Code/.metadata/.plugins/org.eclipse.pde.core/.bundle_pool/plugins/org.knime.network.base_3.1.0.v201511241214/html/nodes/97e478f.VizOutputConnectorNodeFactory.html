Viz Output Connector

Send networks to external program.

Connect a network and execution will send it to the external program.

Supported External Programs

Further information about the supported external programs and a detailed description of this node are available in the Network Visualization section of the Network Mining homepage.

visone

Activate the KNIME connection in the visone console dialog. Networks send will be layouted and displayed immediately.

To use this extension you first have to activate it in the visone options dialog under "extensions".

For further information about visone visit the following link: http://visone.info.

To start visone directly via webstart click here.

Cytoscape

Open the KNIME connector panel via the "Plugins" menu in Cytoscape. The KNIME Connector panel is displayed as a new tab in the Cytoscape Control Panel. Networks send will be layouted and displayed immediately.

To use this extension you first have to install the KNIME connector plug-in in Cytoscape. Please refer to the Network Visualization section of the Network Mining homepage for a detailed installation instruction.

For further information about Cytoscape visit the following link: http://www.cytoscape.org/.

Dialog Options

IP/hostname
The host, the external program is running on. Usually localhost, i.e. this computer.
Port
The port used for the connection (default is 7777).

Ports

Input Ports
0 Network that will be send to the external program.
