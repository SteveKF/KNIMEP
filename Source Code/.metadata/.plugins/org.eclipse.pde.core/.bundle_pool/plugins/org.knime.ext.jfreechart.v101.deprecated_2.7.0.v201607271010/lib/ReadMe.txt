This plugin makes use of the jfreechart and jcommon libraries.

Both libraries are part of the JFreeChart project. 
See http://www.jfree.org. 
It is released under the LGPL (stated at its homepage and its
page at SourceForge http://sourceforge.net/projects/jfreechart).


The sources of the two libraries are included in the directory.
Also, a copy of the LGPL (JFreeChartLicense.txt) can be found 
in the same directory this file is located in. If you are 
missing any of these files, please contact us at 
contact@knime.org.
